# Use rbenv

List all available versions

```shell
rbenv install -l
```

Install Ruby 2.2.1

```shell
rbenv install 2.2.1
```

Uninstall Ruby 2.2.1

```shell
rbenv uninstall 2.2.1
```

See installed versions

```shell
rbenv versions
```

Set global version

```shell
rbenv global 2.2.1
```

Set local version

```shell
rbenv local 2.2.1
```

An alternative is `rvm`.
