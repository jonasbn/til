# Pasting Into Tables

Pasting data into tables in Jira, is unpredictable and often results in errors.

Apparently if you paste into Google Sheets, it will format the data in a way that Jira can understand.

See this [forum post](https://community.atlassian.com/forums/Confluence-questions/how-to-paste-tab-delimited-text-into-a-table/qaq-p/1440805)

It is somewhat cumbersome, but it works.
