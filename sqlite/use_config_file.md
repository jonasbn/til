# Use a Configuration File

You can create a file to set your defaults for the SQLite client in `$HOME/.sqliterc`

Example:

```text
.headers on
.mode box
.prompt "sqlite> " "   ...> "
.nullvalue "<NULL>"
```

Other nifty parts for the config file can be found with these resources:
