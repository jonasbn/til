# use tmux

`tmux` is an awesome tool. I will try to get some of my bookmarked resources collected here. To begin with a nifty [cheatsheet][redditcheatsheet] from Reddit or another [cheatsheet][cheatsheet]. And there is [an interactive one](https://tmuxcheatsheet.org/) available online

I found this great [article][statusbar] on how to customize the `tmux` status bar.

And then there is of course always the [man page][manpage].

The `tmux` plugin manager (tpm) is a necessity. It is installed by cloning the repository. See the TIL entry on [Use Tmux Plugin Manager](../install_tmux_plugin_manager.md).

In can recommend [tmux-sensible](https://github.com/tmux-plugins/tmux-sensible) and resurrect (see: [Use Tmux Resurrect](../use_tmux_resurrect.md)).

For better `tmux` layout and session handling there are tools like:

- [sesh](use_sesh.md)
- [smug](https://github.com/ivaaaan/smug)
- [tmuxinator](https://github.com/tmuxinator/tmuxinator)
- or [tmex](https://github.com/evnp/tmex)

Another interesting tmux related application is **tmuxai**

- [tmuxai](https://tmuxai.dev/getting-started/#tmuxai-layout)
